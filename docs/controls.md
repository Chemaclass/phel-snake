# Controls

| Key | Action |
|-----|--------|
| Arrow keys | Move the snake (player 1 in two-player mode) |
| `W` `A` `S` `D` | Move player 2 (two-player mode only) |
| Space | Accelerate (full version; burst of faster ticks) |
| `P` | Pause / resume |
| `R` | Replay the last run (on the game-over screen) |

Any other key on the game-over screen exits the game.
